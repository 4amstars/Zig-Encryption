# Compile to a standalone executable (optimized for safety)
zig build-exe keygen.zig -O ReleaseSafe -dynamic

# This produces “keygen” in the CWD. Then:
./keygen <size_in_bytes> <password> > key.bin
Or, if you just want to compile & run in one step:

bash
Copy
Edit
zig run keygen.zig -- <size_in_bytes> <password> > key.bin
